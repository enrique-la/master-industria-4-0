# Industria 4.0 — Máster en IA & Data Science

Asignatura de IoT, comunicaciones industriales y ML embebido para la industria manufacturera.

## Contenido

### Comunicaciones Industriales
- Protocolo MQTT y brokers (Mosquitto)
- MIOTY (protocolo IoT de baja potencia)
- Docker + PostgreSQL para almacenamiento en tiempo real

### Internet of Things (IoT)
- Configuración de Raspberry Pi como gateway IoT
- Arduino: programación de microcontroladores
- BLE (Bluetooth Low Energy) para sensores

### Proyecto: Reconocimiento de Actividades con ML
Sistema de reconocimiento de actividades físicas usando sensores IMU en Arduino:
- **Actividades detectadas:** ACOSTARSE, ANDAR, CAER, SENTARSE
- **Pipeline completo:** captura de datos IMU → feature engineering → entrenamiento → despliegue en Arduino
- Clasificador ML embebido en microcontrolador

### Proyecto Gestamp
Implementación IoT en entorno industrial real (colaboración con empresa manufacturera).
